# Story 4.5.3 : Synthèse Tripartite et Dashboard Tuteur

**Status:** Approved

## Story
**As a** Enseignant Tuteur,
**I want** comparer les trois évaluations (Étudiant, Pro, Enseignant) sur une interface unique,
**so that** je peux finaliser la note de stage avec une vision complète.

## Acceptance Criteria
1. Interface de comparaison affichant les 3 séries de notes (sliders juxtaposés ou superposés).
2. Code couleur distinct : Bleu (Étudiant), Orange (Maître de stage), Vert (Tuteur Enseignant).
3. Champ de "Commentaire de synthèse" final par l'enseignant.
4. Calcul automatique d'une note finale suggérée (moyenne pondérée).
5. Bouton de verrouillage final de l'évaluation.
